Deploy on Render.com

1

Fork this Repository

Click the fork button on GitHub to create your own copy of this project.

2

Connect to Render

Go to Render.com and connect your GitHub account, then select this repository.

3

Configure Environment

Set your environment variables and click "Create Web Service".

4

Add Custom Domain

In Render dashboard, add your custom domain and update DNS records.

Render Configuration

# render.yaml
services:
  - type: web
    name: subdomain-creator
    env: node
    plan: free
    buildCommand: npm install
    startCommand: npm start
    envVars:
      - key: NODE_ENV
        value: production
